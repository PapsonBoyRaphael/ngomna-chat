👥 Gestion des Participants

          

            
            
            

              
              
            

            

              
                💡 Seuls les groupes peuvent avoir des participants
                ajoutés/retirés. Vous pouvez entrer plusieurs IDs séparés par
                des virgules (ex: id1, id2, id3).
              
            

          

          

        


        
        

          
🚪 Quitter une Conversation

          

            
            

              
            

            

              
                ⚠️ Cette action est irréversible ! Vous serez retiré du groupe
                et ne recevrez plus les messages. Le créateur du groupe ne peut
                pas quitter (il doit transférer la propriété d'abord).
              
            

          

          

        


        
        

          
✏️ Modifier un Message

          

            
            
            

              
              
            

            

              
                💡 Seul l'auteur du message peut le modifier. Utilisez "Utiliser
                Dernier Message" pour auto-remplir l'ID.
              
            

          

          

        


        
        

          
🗑️ Supprimer un Message

          

            
            
            
            

              
              
            

            

              
                ⚠️ "Pour tout le monde" : seul l'expéditeur peut supprimer, dans
                un délai de 5 minutes.
                

                📌 "Pour moi" : le message reste visible pour les autres.
              
            

          

          

        


        
        

          
📤 Transférer un Message

          

            
            
            

              
              
            

            

              
                📤 Transfère le contenu (texte, fichier, type) du message
                original vers une ou plusieurs conversations.

                💡 Vous pouvez entrer plusieurs IDs de conversations (max 10),
                séparés par des virgules ou un par ligne.

                🔄 Le message transféré sera marqué comme
                transféré avec la référence au message
                d'origine.
              
            

          

          

        


        
        

          
🗑️ Supprimer un Fichier

          

            
            
            

              
            

            

              
                💡 Seul l'uploadeur du fichier peut le supprimer. Si la
                suppression physique est cochée, le fichier sera aussi supprimé
                du disque.
              
            

          

          

        


        
        

          
📥 Télécharger un Fichier

          

            
            

              
            

            

              
                💡 Entrez l'ID du fichier pour le télécharger. Vous pouvez aussi
                télécharger depuis la section "Mes Fichiers".
              
            

          

          

        


        
        

          
📞 Appels Audio / Vidéo


          
          

            
📲 Initier un Appel

            
            
            
            

              
            

            

              
                💡 L'appel crée un message CALL/VIDEO_CALL et notifie le
                destinataire via call:incoming.
              
            

          

          


          
          

            
🎛️ Gérer un Appel en Cours

            
            
            
            

              
              
              
              
            

            

              
                ✅ Décrocher : passe le statut à ANSWERED

                ❌ Refuser : passe le statut à DECLINED

                📴 Raccrocher : termine l'appel (calcule la
                durée)

                📵 Appel Manqué : marque l'appel comme MISSED
